# WFCS 2026 — DALE

Hosts the paper reference page for:
**DALE: DTLS Adaptive Layer Extension for Constrained IoT Networks**
A. S. Mathew, A. Sikora. Proc. IEEE Int. Conf. Factory Commun. Syst. (WFCS), 2026.

## File

- `dale.html` — paper metadata + full reference list. The poster's right-hand QR resolves here.

## Deploy

```
cd "wfcs-2026"
git init
git add dale.html README.md
git commit -m "WFCS 2026 DALE references"
git branch -M main
git remote add origin https://github.com/a-mathew/wfcs-2026.git
git push -u origin main
```

Then in the repo: **Settings → Pages → Source: main branch / root → Save**.

Live URL (~1 min after Pages deploys):
`https://a-mathew.github.io/wfcs-2026/dale.html`

The "Back to profile" link inside `dale.html` points at `https://a-mathew.github.io/anumathew.github.io/`, assuming the reusable profile page lives in the `anumathew.github.io` repo.
